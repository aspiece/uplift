Intro to Java Pacing Guide

Tip: Click a week to expand. The current week is highlighted.
Week 1: Jan 20 – Jan 24 — Getting Started in Java
  • Set up your Java environment
  • Create your workspace
  • Learn the Eclipse IDE or CodeHS
  • Create your first Java class
  • Print statements to the console
  • Complete the Getting to Know You form
Note: Jan 20 is the Start of Semester 2. Choose either Eclipse or CodeHS as your IDE.
Week 2: Jan 27 – Jan 31 — Object-Oriented Programming (OOP) Concepts
  • Understand classes and objects
  • Learn about encapsulation and access modifiers
  • Create and instantiate classes
Week 3: Feb 03 – Feb 07 — Inheritance and Polymorphism
  • Understand inheritance hierarchies
  • Learn about method overriding and polymorphism
  • Project: Animal Class Hierarchy — Create related classes
Note: Feb 6 is a 1/2 Day (ends at 12pm).
Week 4: Feb 10 – Feb 14 — Abstract Classes and Interfaces
  • Learn about abstract classes and methods
  • Implement interfaces and their contracts
  • Design flexible code using abstraction
Note: Feb 11 is Spring Count Day.
Week 5: Feb 17 – Feb 21 — Exception Handling and File I/O
  • Implement try-catch blocks for exception handling
  • Work with file input and output operations
  • Handle errors gracefully in programs
Note: Feb 16 is Staff Professional Development — NO SCHOOL for students.
Note: Feb 20 is 100th Day of School — 1/2 Day (ends at 12pm).
Week 6: Feb 24 – Feb 28 — Collections Framework (ArrayList, HashMap)
  • Work with ArrayList and other collection types
  • Understand HashMap for key-value storage
  • Iterate through collections efficiently
Week 7: Mar 03 – Mar 07 — Recursion and Problem Solving
  • Understand recursive functions and base cases
  • Solve problems using recursion
  • Compare recursion vs iteration
Note: Mar 6 is a 1/2 Day (ends at 12pm).
Week 8: Mar 10 – Mar 14 — Sorting and Searching Algorithms
  • Implement bubble sort and selection sort
  • Understand linear and binary search algorithms
  • Analyze algorithm efficiency and Big O notation
Week 9: Mar 17 – Mar 21 — Capstone Project Planning & Design
  • Plan your semester capstone project
  • Design class structures and requirements
  • Create project documentation and design patterns
Note: Mar 18 is Parent-Teacher Conferences.
Note: Mar 20 is a 1/2 Day (ends at 12pm).
Week 10: Mar 24 – Mar 28 — Capstone Project Development
  • Begin implementing your capstone project
  • Code core functionality and classes
  • Debug and test your implementation
Week 11: Mar 31 – Apr 04 — Spring Break (No Classes)
  • Enjoy Spring Break — No classes this week
  • Optional: Continue capstone project work if desired
Spring Break (Mar 30–Apr 03) — NO SCHOOL
*** NO ASSIGNMENTS DUE THIS WEEK ***
Week 12: Apr 07 – Apr 11 — Capstone Project Continuation
  • Resume capstone project after Spring Break
  • Complete core features and functionality
  • Implement user interface enhancements
Week 13: Apr 14 – Apr 18 — Advanced Features and Optimization
  • Add advanced features to your capstone project
  • Optimize code for performance
  • Refactor and improve code quality
Note: Apr 17 is a 1/2 Day (ends at 12pm).
Week 14: Apr 21 – Apr 25 — Project Finalization and Documentation
  • Finalize all capstone project features
  • Write comprehensive project documentation
  • Create user guides and code comments
Week 15: Apr 28 – May 02 — Testing and Quality Assurance
  • Thoroughly test your capstone project
  • Identify and fix bugs
  • Ensure all features work as intended
Note: May 1 is a 1/2 Day (ends at 12pm).
Week 16: May 05 – May 09 — Project Presentations and Peer Reviews
  • Prepare presentation of your capstone project
  • Present your project to the class
  • Review and provide feedback on peer projects
Note: May 8 is NO SCHOOL.
Week 17: May 12 – May 16 — NWEA Testing and Course Review
  • Participate in NWEA Spring Testing
  • Review key Java concepts and best practices
  • Discuss course highlights and learning outcomes
Note: May 15 is a 1/2 Day (ends at 12pm).
Note: May 18–22 is NWEA Spring Testing Week.
Week 18: May 19 – May 23 — Extended NWEA Testing Week
  • Continue NWEA Spring Testing
  • Advanced Java Topics and Reflection
  • Explore additional libraries and frameworks
Note: May 25 is NO SCHOOL — Memorial Day.
Week 19: May 26 – May 30 — Final Project Reviews and Feedback
  • Receive final feedback on your capstone project
  • Make final revisions based on feedback
  • Prepare final project submission
Note: May 29 is the Last Day for Graduating Seniors.
Week 20: Jun 02 – Jun 06 — End of Year and Graduation
  • Final submissions and course wrap-up
  • Course reflections and feedback
  • Awards and recognition
Note: Jun 5 is LAST DAY of School Year — 1/2 Day (ends at 12pm).
Note: Jun 6 is Graduation for Seniors.

Intro to Java: Student Resource Hub

Follow these steps to get started in CodeHS, complete your first assignment (MadLibs), and submit your work to Accellerate.

🚀Guided Setup

Step 1 of 9

Step 1: Create your CodeHS account

Set up your free account first so you can access the Java sandbox.

👉 Create your free CodeHS account

Step 2: Watch CodeHS setup video

Watch this short video to learn how to navigate CodeHS quickly.

Step 3: Open the Java sandbox

Open Sandbox and make sure Java is selected for your coding environment.

👉 Open CodeHS Sandbox

Step 4: Open the MadLibs starter file

Open the starter-code page, then select the MadLibs starter link.

👉 Open starter code page (MadLibs)

Step 5: Copy starter code into a new program

After opening the starter file, copy all code and place it into your own new program.

  • Select all code in the starter file and copy it.
  • Start a new Java program in CodeHS and name the file MadLibs.java.
  • Paste the copied code into the new program file.
  • Run the program to confirm it works before editing.

Step 6: Use the MadLibs walkthrough

Watch this if you need help completing the assignment.

Step 7: Download your .java file from CodeHS

In CodeHS, open your completed MadLibs.java file.

  • Select More in the top-right corner.
  • Select Download.
  • Choose the .txt version of the file.
  • Verify the file downloaded before moving on.

Step 8: Submit to Accellerate

Upload your downloaded MadLibs file to Accellerate and submit the assignment.

Step 9: More starter code

All assignment starter links are now in one place.

👉 Open Assignment Starter Code page

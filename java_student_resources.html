Intro to Java: CodeHS Setup & Submission Guide

Follow this guided workflow to set up CodeHS, complete MadLibs, download your file, and submit in the course.

Overview

By the end of this guide, you will:

  • Set up your free CodeHS account
  • Access and complete the MadLibs Java assignment
  • Download your completed MadLibs.java file
  • Submit your work to the course

No prior experience required. If you get stuck, see the help section at the end!

Step 4: Open the MadLibs starter file

Open the starter-code page, then select the MadLibs starter link.

👉 Open starter code page (MadLibs) ↗ (opens in new tab)

Step 5: Copy starter code into a new program

After opening the starter file, copy all code and place it into your own new program.

  • Select all code in the starter file and copy it.
  • Start a new Java program in CodeHS and name the file exactly MadLibs.java (case-sensitive, no spaces).
  • Paste the copied code into the new program file.
  • Run the program to confirm it works before editing.
Tip: Double-check your file is named MadLibs.java before moving on.

Step 6: Use the MadLibs walkthrough

Watch this if you need help completing the assignment.

Step 7: Download your MadLibs.java file from CodeHS

In CodeHS, open your completed MadLibs.java file.

  • Select More in the top-right corner.
  • Select Download.
  • Choose the .txt version of the file.
  • Verify the file downloaded before moving on. Check that the file is named MadLibs.java.txt or MadLibs.java and contains your code.

Step 8: Submit to the course

Upload your downloaded MadLibs.java file to the course and submit the assignment.

  • Double-check your file is named MadLibs.java (not .txt unless required by your instructor).
  • Open the file and confirm it contains your completed code.
  • After uploading, review your submission:
    • Did you upload the correct file?
    • Is it named MadLibs.java?
    • Does it contain your finished code?
